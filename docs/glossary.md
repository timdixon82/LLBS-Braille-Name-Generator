# Glossary: LLBS Braille Name Generator

## Braille

A system of reading and writing by touch, using patterns of raised dots on a page. Braille was invented by Louis Braille in the nineteenth century and is used worldwide by blind and partially sighted people.

## Grade 1 braille

Grade 1 braille, also called uncontracted braille, represents each letter of the alphabet as a distinct dot pattern. Every letter is spelled out individually, with no abbreviations or contractions. Grade 1 is the starting point for learning braille and is used in this tool.

## Grade 2 braille

Grade 2 braille uses a system of contractions and shortened forms to make braille faster to read and write. This tool does not produce Grade 2 braille. Grade 2 is covered by the separate Braille-Reference project.

## Braille cell

The basic unit of braille. A cell consists of six dot positions arranged in two columns of three. The left column contains positions 1 (top), 2 (middle), and 3 (bottom). The right column contains positions 4 (top), 5 (middle), and 6 (bottom). A filled dot position is raised; an empty position is flat.

## UEB

Unified English Braille (UEB). The international standard for English braille adopted by the UK in 2011. This tool uses UEB Grade 1.

## LLBS

Lincoln and Lindsey Blind Society. A charity that has supported people with vision impairment in Greater Lincolnshire for over 100 years. The organisation is the subject of this tool and holds the copyright on its logo.

## Canvas

The HTML Canvas element (`<canvas>`). A browser API that lets JavaScript draw graphics programmatically. This tool uses a hidden 1080 by 1080 pixel canvas to compose the braille image before exporting it as a PNG file.
